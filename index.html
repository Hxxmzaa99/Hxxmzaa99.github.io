Partie 3 : Formulaire en HTML

  • name Le nom du formulaire. Il doit être unique parmi tous les formulaires d'un document, et ne doit pas être une chaîne de caractères vide.
  • action L'URL qui traite l'envoi du formulaire. Cette valeur peut être remplacée par un attribut formaction sur un boutton.

Voici les définitions des différents types d'input que vous utiliserez :
Checkbox : Permet de créer une case à cocher, qui peut être sélectionnée ou désélectionnée par l'utilisateur pour indiquer un choix binaire (oui/non).
Number : Permet à l'utilisateur d'entrer une valeur numérique. Ce champ inclut généralement des flèches pour augmenter ou diminuer la valeur.
Password : Crée un champ pour entrer un mot de passe. Le texte saisi est masqué par des points ou des astérisques pour protéger la confidentialité.
Range : Crée un curseur qui permet à l'utilisateur de sélectionner une valeur dans une plage définie, souvent utilisée pour des paramètres comme le volume ou la luminosité.
Search : Conçu pour les entrées de recherche, ce champ de texte peut afficher un bouton pour effacer le texte ou offrir des suggestions automatiques dans certains cas.
Submit : Un bouton qui, lorsqu'il est cliqué, envoie les données du formulaire au serveur pour traitement.
